1 div <div> </div> El <div> define una división o una sección en un documento HTML El <div> se utiliza como un contenedor para elementos HTML, los cuales luego se pueden estilizar con CSS o manipular con JavaScript. 
<div class="contenedor">
    <p>Este es un párrafo dentro de un div.</p>
</div>

Este es un párrafo dentro de un div.
2 table <table> </table> La etiqueta <table> define una tabla HTML. La etiqueta <table> se utiliza para crear tablas en HTML. Dentro de una tabla, se utilizan otras etiquetas como <tr> para definir filas, <th> para definir encabezados de tabla, y <td> para definir celdas. 
<table border="1">
        <tr>
            <th>Encabezado 1</th>
            <th>Encabezado 2</th>
        </tr>
        <tr>
            <td>Celda 1</td>
            <td>Celda 2</td>
        </tr>
    </table>
Encabezado 1 Encabezado 2
Celda 1 Celda 2
3 li <li> </li> Define un ítem en una lista. La etiqueta <li> se utiliza para definir un elemento en una lista ordenada (<ol>) o desordenada (<ul>). 
<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>
  • Elemento 1
  • Elemento 2
4 ul <ul> </ul> Define una lista desordenada. La etiqueta <ul> se utiliza para crear una lista desordenada. Los elementos de la lista se definen usando <li>. 
<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>
  • Elemento 1
  • Elemento 2
5 input <input /> Define un campo de entrada de datos. La etiqueta <input> se utiliza para crear campos interactivos en formularios web. Puede ser de varios tipos, como text, password, submit, etc. 
<input type="text" placeholder="Ingrese texto" />
6 button <button> </button> Define un botón clickable. La etiqueta <button> se utiliza para crear un botón clickable en un formulario o una interfaz web. 
<button>Click Me</button>
7 a <a> </a> Define un hipervínculo. La etiqueta <a> se utiliza para crear enlaces a otras páginas web, direcciones de correo electrónico, o cualquier otro URL. El atributo href especifica la URL del enlace. 
<a href="https://www.ejemplo.com">Visita Ejemplo</a>
 Visita Ejemplo
8 p <p> </p> Define un párrafo. La etiqueta <p> se utiliza para agrupar bloques de texto en párrafos. Los navegadores añaden un espacio antes y después de cada párrafo por defecto. 
<p>Este es un párrafo.</p>

Este es un párrafo.
9 span <span> </span> Define una sección en línea. La etiqueta <span> se utiliza para agrupar elementos en línea para propósitos de estilo o manipulación con scripts. 
<span style="color: red;">Texto en rojo</span>
 Texto en rojo
10 video <video> </video> Define un video incrustado. La etiqueta <video> se utiliza para incrustar videos en HTML. Puede incluir atributos como controls, autoplay, y loop. 
<video controls>
    <source src="video.mp4" type="video/mp4">
    Tu navegador no soporta el elemento de video.
</video>
11 audio <audio> </audio> Define un audio incrustado. La etiqueta <audio> se utiliza para incrustar archivos de audio en HTML. Puede incluir atributos como controls, autoplay, y loop. 
<audio controls>
    <source src="audio.mp3" type="audio/mp3">
    Tu navegador no soporta el elemento de audio.
</audio>
12 img <img /> Define una imagen. La etiqueta <img> se utiliza para incrustar imágenes en una página web. El atributo src especifica la URL de la imagen y el atributo alt proporciona un texto alternativo. 
<img src="imagen.jpg" alt="Descripción de la imagen" />
13 form <form> </form> Define un formulario HTML. La etiqueta <form> se utiliza para crear un formulario HTML para la entrada del usuario. Puede incluir varios campos de entrada, botones, etc. 
<form>
    <input type="text" name="nombre" placeholder="Nombre">
    <input type="submit" value="Enviar">
</form>
14 ol <ol> </ol> Define una lista ordenada. La etiqueta <ol> se utiliza para crear una lista ordenada. Los elementos de la lista se definen usando <li>. 
<ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ol>

  2. Elemento 2
15 body <body> </body> Define el cuerpo del documento. La etiqueta <body> se utiliza para contener todo el contenido de un documento HTML, como texto, imágenes, tablas, etc. 
<body>
    <p>Contenido del cuerpo</p>
</body>

Contenido del cuerpo
16 h1,h2,h3,h4,h5 <h1> </h1>, <h2> </h2>, <h3> </h3>, <h4> </h4>, <h5> </h5> Define encabezados de diferentes niveles. Las etiquetas <h1> a <h5> se utilizan para definir encabezados, con <h1> siendo el más importante y <h5> el menos importante. 
<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>

17 script <script> </script> Define un script embebido. La etiqueta <script> se utiliza para incluir código JavaScript en un documento HTML. Puede incluirse en el <head> o en el <body>. 
<script>
    console.log('Hola Mundo');
</script>

18 style <style> </style> Define estilos CSS embebidos. La etiqueta <style> se utiliza para incluir CSS en un documento HTML. Generalmente se coloca dentro de la sección <head>. 
<style>
    body { font-family: Arial; }
</style>
 (Este documento ya incluye estilos en el <head>)
19 link <link /> Define una relación con un recurso externo. La etiqueta <link> se utiliza para vincular documentos externos, como hojas de estilo CSS. Generalmente se coloca en el <head>. 
<link rel="stylesheet" href="estilos.css" />
 (No hay una visualización directa, pero vincula el documento CSS)
